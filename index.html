Lumora - Project Overview

Lumora is a progressive web application, responsively designed for both mobile and desktop, and built with React, TypeScript, and Redux Toolkit. It enables users to write and manage journal entries online with ease across multiple devices. Users can create personal accounts, track their self-care journey through a journaling streak, and receive a new daily affirmation. Firebase handles user authentication, database storage, and hosting, ensuring a secure and reliable experience.

Features:

  • Login / Signup: Users are able to login or signup with their personal account.
  • Dashboard: Overview with daily affirmation, journaling streak, past journal entries and favorite affirmations.
  • Journal entries: Users can view, edit and delete their past journal entries.
  • Favorite affirmations: Users can see a list of their favorite affirmations and remove them from favorites.
  • Account: Users are able to view and update their account data or delete their account entirely.

Technologies:

  • Tailwind CSS

Eventory - Project Overview

Eventory is a serverless progressive web application built with React, developed using a test-driven development approach. The Google Calendar API is utilized to display upcoming events in various cities, offering offline functionality, data visualization, and a responsive, cross-platform user experience.

Features:

  • Authentication: Users authentication via Google Calendar API and OAuth2 authentication flow.
  • Filter Events by City: Users can search for events in specific cities and show/hide event details as needed.
  • Event Limitation: Specify the number of events displayed to suit user preferences.
  • Data visualization: Interactive charts provide insights into event genres and locations.
  • Offline Functionality: The app works offline or in slow network conditions using local storage and a service worker.
  • App Installation: Users can add the app to their home screen on mobile or install it on desktop.
  • Offline monitoring tool: Integrated with Atatus for monitoring and optimizing app performance.

Technologies:

  • React
  • Recharts
  • AWS Lambda
  • Service Workers
  • Test-Driven Development (TDD)
  • Jest

ChatterBox - Project Overview

ChatterBox is a mobile chat app built with React Native and Expo, allowing users to exchange messages, share images, and send their location in real-time. The app integrates Firebase for anonymous authentication, real-time data storage, and offline message access to ensure a seamless chatting experience.

Features:

  • User Customiszation: Users can enter their name and choose a background color for their chat screen before starting a conversation.
  • Messaging Interface: The app provides a chat interface with an input field and send button for exchanging messages.
  • Media Sharing: Users can send images directly from their device's gallery or take photos with the device's camera.
  • Location Sharing: Real-time location sharing is supported, displayed in the chat as a map view.
  • Offline Functionality: Conversations are stored both locally and online to ensure functionality even when offline.

Technologies:

  • React Native
  • Expo
  • Gifted Chat
  • Firebase Firestore
  • Firebase Cloud Storage

Pokédex - Project Overview

Pokédex is an interactive JavaScript web application that that allows users to explore a list of Pokémon fetched from an external API. The app features a responsive design styled with Bootstrap, ensuring a seamless experience across all devices. Users can view detailed information about each Pokémon, including its type, dimensions, and abilities.

Rock Paper Scissors - Project Overview

Rock Paper Scissors is an interactive JavaScript web application that lets users play the classic Rock Paper Scissors game against the computer, featuring dynamic visuals and real-time score tracking.

  • Choose your Move: Players can pick Rock, Paper, or Scissors to compete against the computer.
